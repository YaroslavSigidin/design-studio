# Согласовано — отдельный репозиторий студии

Статический сайт дизайн-студии `Согласовано`, вынесенный в отдельный GitHub-репозиторий и готовый к публикации через GitHub Pages.

## Что внутри

- `index.html` — главная страница студии
- `case.html` — шаблон отдельного кейса
- `data/cases.manifest.json` — все данные кейсов для карточек и страниц
- `assets/` — стили, скрипты, иконки, видео и изображения кейсов

Сайт автономный: больше не зависит от корня портфолио и может деплоиться как отдельный проект.

## Локальный запуск

```bash
python3 -m http.server 8766 --bind 127.0.0.1
open http://127.0.0.1:8766/
```

Или двойной клик по `open.command`.

## Деплой

Репозиторий настроен на GitHub Pages через GitHub Actions.

После пуша в `main` деплой запускается автоматически workflow-файлом:

```text
.github/workflows/deploy.yml
```

## Примечание по обновлению кейсов

Текущий `data/cases.manifest.json` уже содержит контент кейсов и ссылки на нужные изображения, которые включены в этот репозиторий.
