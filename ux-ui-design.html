Вопросы

До начала работ

Можно заказать только UX?

Да. Можно отдельно проработать архитектуру, пользовательские сценарии и кликабельный прототип для проверки гипотез.

Работаете с существующей командой разработки?

Да. Учитываем текущий стек и компоненты, проводим передачу макетов и остаёмся на связи по вопросам реализации.

Подходит ли услуга для MVP?

Да. Для MVP фиксируем минимальный набор сценариев и проектируем систему так, чтобы её можно было расширять после проверки спроса.
